Add / manage staff credentials

In Netlify, go to Site settings → Identity → Users and click Invite users. Create individual staff accounts (recommended). Assign roles:

  • staff → can use Staff Lookup
  • admin → can access this Admin page + Staff Lookup

Operational control: if an employee leaves, disable their user instantly.

Update inventory + offers (current workflow)

Inventory and offers are currently stored in your GitHub repo. Best practice is to migrate to Google Sheets or a small admin CMS so staff can update without touching GitHub.

  • Inventory (private): /private/products_private.csv
  • Inventory (public): /products_public.csv (no price/qty)
  • Offers: /offers.csv
